Nested Drag and Drop

A picture says more than thousand words. We have prepared a screencast of the desired result. Have a look at it now.

Task

Create 2 "classes":

  • One class (container) creates draggable containers
  • Other class (node) creates draggable nodes that can be dropped in instances of the latter

Once an instance of node is dropped on an instance of container the node rests in the container and moves together with the container.

When removed from the container the node will move independently again.

Prioritized goals

Must have

  • Everything is generated programmatically
  • "containers" are draggable
  • "nodes" are draggable and can be dropped in instances of container
  • Once a node is dropped on a container it becomes part of the container and "follows" its new parent
  • Instances of node are detachable from the container

On best effort

  • Allow dropping nodes from one container to the other
  • Visually indicate valid drop targets. Highlight or set a different border when the dragged node enters a valid drop target
  • Add some eye candy and make it look nice
  • Try to have some fun with the task ;)

Constraints

  • You may use any framework you want however we expect that you keep the dependencies to a minimum.
  • Use the DOM (HTML/SVG). Do NOT use canvas
  • Do NOT webpack a simple thing like this. It must run without having to trigger any build jobs or unnecessary scaffolding.
  • No markup allowed. Everything is meant to be created programmatically
  • Time: Spend no longer than 4 hours. We also accept partial solution as long as you tackle aspects of the task in ascending order. MUST HAVES come first.

We hope you have some fun with this task.
